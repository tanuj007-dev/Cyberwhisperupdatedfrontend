# Video assets for deployment (Vercel etc.)

Place your video files here so they are served correctly in production. Use **no spaces** in filenames.

- **cyber-whisper-demo.mp4** – Used by the "Watch Demo" button (hero section and B2B page). If your file is named `Cyber Whisper.mp4`, rename or copy it to `cyber-whisper-demo.mp4`.
- **enterprise-fortress.mp4** – Optional: B2B hero section background video.
- **cyber-promo.mp4** – Optional: Training section promo video.

These paths are URL-safe and work on Vercel and other static hosts.
